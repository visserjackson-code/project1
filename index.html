This page serves as a tracker for a "Nuzlocke" style run of Pokémon Sapphire. While you can add and modify rules at your leisure, a Nuzlocke run has two main rules.

  1. You are only allowed to catch the first Pokémon you encounter on each route.
  2. If a Pokémon faints in battle, that Pokémon is considered "dead" and can no longer be used for the remainder of the run
For more information and additonal rule ideas, see the Wikipedia article here.
